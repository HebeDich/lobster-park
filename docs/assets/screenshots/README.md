# Screenshot Placeholders

当前目录用于放 README 中展示的项目截图。

目前 README 已经切换到真实 PNG 截图。

## 当前使用的文件名

- `docs/assets/screenshots/instance-overview.png`
- `docs/assets/screenshots/quick-setup.png`
- `docs/assets/screenshots/channel-integration.png`
- `docs/assets/screenshots/platform-chat.png`
- `docs/assets/screenshots/instance-terminal.png`

## 建议截图内容

- `instance-overview.png`：实例概览页，包含启动、配置、进入对话等主操作
- `quick-setup.png`：快速配置页，体现模型配置和完成向导
- `channel-integration.png`：渠道接入页，体现渠道选择与测试
- `platform-chat.png`：平台对话页，体现平台内聊天与模型回复
- `instance-terminal.png`：实例终端页，体现真实 PTY / 容器终端能力

## 替换方式

1. 直接用新的真实图片覆盖同名文件
2. 或者把真实截图放到新路径后，更新 `README.md` 中的引用
