# Product History Archive

该目录保留项目早期的需求、交付、评审和 OpenAPI 历史文档。

## 说明

- 这些文档用于保留项目演进脉络，不属于运行时代码
- 当前公开仓库将它们从根目录迁移到归档区，以减少主目录噪音
- 部分历史计划文档中仍可能引用旧的根目录路径；如遇到这类引用，请在原路径前补上 `docs/archive/product-history/`

## 当前仍被构建链使用的文件

- `docs/archive/product-history/LobsterPark_Docs_V1_4/LobsterPark_OpenAPI_V1_4.yaml`

该 OpenAPI 文件仍用于前端 API 代码生成。
